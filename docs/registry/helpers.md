# Helper Registry

Living summary of all helper classes. Update this file when creating, updating, or deleting a helper.

| Helper            | Class                         | Path                               | Purpose                                         |
| ----------------- | ----------------------------- | ---------------------------------- | ----------------------------------------------- |
| Date & Time       | `DateTimeHelper`              | `@helpers/date-time-functions`     | Format, add/subtract, compare dates             |
| Data Generator    | `DataGenerator`               | `@helpers/generate-data-functions` | Random emails, names, numbers, strings          |
| File              | `FileHelper`                  | `@helpers/generate-data-functions` | Clone, exists, delete files                     |
| Array / String    | `ArrayHelper`, `StringHelper` | `@helpers/helper-functions`        | Collection utilities, string manipulation       |
| Excel             | `ExcelHelper`                 | `@helpers/excel.helper`            | Open, read/write cells, rows/columns as JSON    |
| PDF               | `PdfHelper`                   | `@helpers/pdf.helper`              | Text, tables, metadata, forms, visual snapshots |
| Word              | `WordHelper`                  | `@helpers/word.helper`             | Text, headings, tables, images, templates       |
| Database          | `DatabaseHelper`              | `@helpers/database.helper`         | Query, insert, delete for test data cleanup     |
| Test Data         | `TestDataHelper`              | `@helpers/test-data.helper`        | Factory + auto-cleanup fixture for test state   |
| Schema Validation | `ValidateSchemaHelper`        | `@helpers/validate-schema.helper`  | JSON schema validation for API responses        |
| Accessibility     | `AccessibilityHelper`         | `@helpers/accessibility.helper`    | Axe-core scans, violation reporting             |
